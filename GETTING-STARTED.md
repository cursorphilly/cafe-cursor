# Getting started with Cursor

First Cafe Cursor? You only need a laptop and curiosity — but having Cursor installed before you arrive saves setup time.

## Install Cursor

1. Download from [cursor.com](https://cursor.com)
2. Sign in and open a folder (any project, or fork a [starter](starters/README.md))
3. Open **Chat** or **Agent** in the sidebar

## Quick orientation

| Action | How |
|--------|-----|
| Ask about code | Select a file, open Chat, ask a question |
| Edit multiple files | Use Agent mode with a clear goal |
| Project rules | Add `.cursor/rules/` or `AGENTS.md` at repo root |
| Skills | Copy from [`skills/`](skills/) into your project's `.cursor/skills/` |
